1. Introduction



Predicting a movie's success poses a significant challenge in the entertainment field. This is especially relevant because production and marketing costs are high, so accurate decisions must be made regarding budgeting, production, distribution, and advertising. Poorly performing films can result in major financial losses, while successful films can generate billions of dollars in revenue and achieve long-term franchise growth. By identifying the factors most associated with successful films, production companies and stakeholders can make more informed decisions regarding the types of movies they choose to invest in. Our investigation specifically centers around the questions: Does a larger budget, primary genre, and other features necessarily lead to higher movie revenue and higher audience ratings? Through statistical analysis, exploratory data analysis, and machine learning models applied to The Movies Dataset from Kaggle, we aim to better understand the relationships between different movie characteristics and how they correlate with a movie’s success, while also evaluating how effectively these outcomes can be predicted from the metadata alone.


Dataset source: Banik, R. (2017). The Movies Dataset. Kaggle. https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset

This Movies Dataset is a large-scale movie analytics dataset. It consists of metadata for over 45,000 films and over 26 million user ratings. Most of these movie ratings are collected from popular movie rating services sucha s TMDb and MovieLens, though it's important to note that Rotten Tomatoes is not included in this data, which is perhaps the most well known service.

It includes information through features such as genres (lists of 2-3 for most movies), budgets, revenues, release dates, cast + crew data, production companies, user rating behaviour, brief summary/overviews, and more, making it quite extensive in information quantity. It also icludes relational files in credits.csv, keywords.csv, and ratings.csv to give further context and information for each movie, allowing users of this data to connect multiple dimensions of overall movie information for more substanced and nuanced analyses of movie trends.


Some elementary domain knowledge required for this field involves understanding the relative earnings of movies correlated to budget. Every now and then, we hear about the newest highest grossing movies, like the original Avatar, Avengers: Endgame, and, if you're a fan of international films, the infamous Ne Zha. The general trend is that, the more budget is thrown into the movie, the more it generates. Recently, however, major exceptions have shown up. A lot of studios recently have been throwing money at intriguing projects, such as Sony, for diminishing returns. For example, the movie "Morbius" effectively made a LOSS of 300 million, and that's before you even consider marketing budget. For a general rule of thumb, the average studio spends the same money MAKING the movie as MARKETING it, so the TRUE budget of a movie is usually double the listed budget for financials. For the sake of simplicity, we will not be representing that in this dataset, as we want to stay as true to the original dataset as possible.

# method to adjust for inflation using CPI-U data, converting nominal amounts to 2026 dollars
def adjust_for_inflation(amount, release_year):
    """Convert a nominal dollar amount to TARGET_YEAR (2026) dollars using CPI-U.
    Returns NaN for missing, zero, or out-of-table values."""
    if pd.isna(amount) or amount == 0:
        return float('nan')
    cpi_then = CPI_BY_YEAR.get(int(release_year)) if not pd.isna(release_year) else None
    if cpi_then is None:
        return float('nan')
    return amount * (CPI_TARGET / cpi_then)

def safe_parse(val):
    """Safely convert a stringified Python list/dict back to a Python object,
    returning [] on any failure like NaN."""
    try:
        return ast.literal_eval(val)
    except (ValueError, SyntaxError, TypeError):
        return []

def extract_names(val):
    """Extract every 'name' field from a stringified list of dicts."""
    parsed = safe_parse(val)
    if isinstance(parsed, list):
        return [d['name'] for d in parsed if isinstance(d, dict) and 'name' in d]
    return []

def extract_first_name(val):
    """Return only the first 'name' from a stringified list of dicts,
    returning 'Unknown' if none is found"""
    names = extract_names(val)

# [Runtime: Missing at Random (MAR)]
# During our EDA, we noticed that runtime has 0.6% null values (~263 rows).
# We classify this as MAR (missing at random)) because missingness is likely related to
# other observed variables — specifically, films from more obscure, indie developers
# or foreign markets tend to have less complete metadata on large datasets such as this.

# Because missingness correlates with genre, we decided that a global median fill would
# introduce bias. For example, filling a missing Documentary runtime with the
# overall median of ~95 mins would be inaccurate when documentaries typically
# run shorter. Instead, we use genre-median imputation each missing runtime
# is filled with the median runtime of all other films sharing the same
# primary_genre. Alongside our analysis of movies based on genre,
# we believe that this approach preserves the natural runtime distribution
# within each genre and produces more realistic imputations.

# Derive primary_genre inline if it doesn't exist yet

    df['primary_genre'] = df['genres'].apply(extract_first_name)

df['runtime'] = df.groupby('primary_genre')['runtime'].transform(
    lambda x: x.fillna(x.median())
)

- Despite this, with an R-squared of 0.707, we find release year alone explains about 70.7% of the variation in mean adjusted revenue, which we noted as a surprisingly strong linear relationship for a single predictor given the result.
- We believe the remaining 29.3% of variation is likely driven by external factors such as the globalization of the film market towards the latter end of the 20th century, the rise of franchise and blockbuster culture in the early 2010s, improved international distribution throughout history, and shifting audience behavior due to socioeconomic developments and other worldly events, factors which release year alone cannot be expected to capture.
